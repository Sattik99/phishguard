# Threat Model

## Threats Covered

### Email Spoofing

Attackers impersonating legitimate senders.

### Credential Harvesting

Emails designed to steal usernames and passwords.

### Business Email Compromise (BEC)

Attempts to deceive users into performing sensitive actions.

### Malicious Links

Links directing victims to attacker-controlled infrastructure.

### Social Engineering

Urgency-based manipulation techniques.

---

## Threats Not Covered

### Zero-Day Malware

The project does not analyse executable behaviour.

### Sandbox Analysis

Attachments are not dynamically executed.

### Advanced Threat Actors

Highly sophisticated phishing techniques may evade rule-based detection.

### Machine Learning Evasion

The project currently uses a rule-based approach.
